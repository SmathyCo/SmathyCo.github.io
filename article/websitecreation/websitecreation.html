How to code a website?

Installing A Code Editor

1. Download the code editor called "Visual Studio Code" from here.

2. Run the downloaded file.

3. Follow the installation instructions.

4. Run the code editor.

Creating A New Project

1. Open the file explorer.

2. Create a new folder.

3. Drag and drop the folder in the code editor.

4. On the top left you should see a "+" when you hover over your project's folder name.

5. Name your file "index.html".

6. Click on the file to open the "index.html" file in the code editor.

Writing HTML Code

1. In the "index.html" file, write the following code:


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        
    </body>
</html>

Explanation: This code represents a basic HTML5 document structure. It includes:

  • <!DOCTYPE html>: Declares the document type and HTML version.
  • <html lang="en">: The root element of the HTML document, with the language set to English.
  • <head>: Contains meta-information about the document, such as character encoding, viewport settings, and the title.
  • <meta charset="UTF-8">: Specifies the character encoding for the document.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Sets the viewport to make the page responsive on different devices.
  • <title>Document</title>: Sets the title of the document, which appears in the browser tab.
  • <body>: Contains the content of the document, such as text, images, and other elements.

    • Styling Your Website

    1. To start adding style to your website we are going to use some CSS (Cascading Style Sheet).

    2. Add a <style></style> into your <body></body> tag.

    3. Between the two style tags you can now add some CSS code. To start lets add this:

    
/* This CSS targets both <body> and <html> elements */
body, html {
    /* Centers the element horizontally and removes margins on the sides */
    margin: auto 0;
    
    /* Sets the background color to blue */
    background-color: blue;
}

    Viewing Your Website

    1. Open the file explorer.

    2. Open your projet folder.

    3. Right click your "index.html" file and click "Open With" then choose with your browser.

    4. Your website should now be displayed in your browser.

    Conclusion

    That's it! You've now learned how to create a basic website using HTML and CSS. You can now start building more complex and interactive websites. Happy coding!
