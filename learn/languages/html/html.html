What is HTML?

HTML, or HyperText Markup Language, is the backbone of the World Wide Web. It's a markup language used to create and structure web pages by defining the content and layout of a document. HTML consists of a series of elements, each enclosed in angled brackets, which describe different parts of a webpage, such as headings, paragraphs, images, links, and more.

In essence, HTML serves as the skeleton of a webpage, providing the structure and organization necessary for presenting content on the internet. It's a fundamental language for anyone looking to create or understand the web.

HTML Basics Tutorial

Welcome to the HTML Basics Tutorial by SmathyCo! In this tutorial, you'll learn the fundamentals of HTML (Hypertext Markup Language), which is the standard markup language for creating web pages.

Step 1: Understanding HTML Structure

HTML documents are structured using elements, which are represented by tags enclosed in angle brackets. The basic structure of an HTML document consists of the <html>, <head>, and <body> tags.

The <!DOCTYPE html> declaration at the beginning of the document specifies the version of HTML being used. The <html> tag is the root element of the document, and it contains all other HTML elements.

The <head> tag contains meta-information about the document, such as the character encoding and page title. The <body> tag contains the visible content of the document, such as text, images, and links.


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>

Step 2: Adding Content with Tags

HTML tags are used to define different types of content, such as headings, paragraphs, links, images, and more. Here are some commonly used tags:

  • <h1> - <h6>: Headings - Used to define headings of different levels.
  • <p>: Paragraphs - Used to define paragraphs of text.
  • <a>: Links - Used to create hyperlinks to other web pages or resources.
  • <img>: Images - Used to embed images into the document.
  • <ul> and <ol>: Unordered and Ordered Lists - Used to create lists of items.
  • <li>: List Items - Used to define items in a list.

Step 3: Structuring Content with Elements

HTML elements can be nested inside each other to create a structured hierarchy of content. For example, you can place headings, paragraphs, and lists inside a <div> element to create sections of content.

Step 4: Adding Attributes

HTML attributes provide additional information about elements and are added to the opening tag of an element. Common attributes include class, id, src, href, and alt.

For example, the src attribute specifies the URL of an image, and the alt attribute provides alternative text for the image.


<img src="example.jpg" alt="Example Image">

Step 5: Testing Your HTML

After writing your HTML code, save the file with a .html extension and open it in a web browser to see how it looks. You can also use online HTML validators to check for errors in your code.

That's it for this HTML Basics Tutorial! With these fundamentals, you're ready to start building your own web pages. Happy coding!
